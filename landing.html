Architecture

EventBridge (rate: 1 hour)
↓
AWS Lambda ← triggers stock ingestion
↓
AWS EC2 (Ubuntu 22.04) ← FastAPI application server
↓            ↓
SQLite DB    AWS S3 ← persistent database backups

Endpoints

GET/stock/{ticker}Fetch live stock price and store to DB
GET/history/{ticker}Retrieve full price history for a ticker
GET/analysis/{ticker}Trend analysis — min, max, avg, % change
GET/chart/{ticker}Time-series price chart (PNG)
GET/docsInteractive Swagger API documentation
